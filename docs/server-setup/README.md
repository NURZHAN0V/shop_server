# Создание сервера на Express.js

Полное руководство по созданию сервера на основе текущего проекта.

## Структура документации

1. **[Инициализация проекта](./01-initialization.md)** — создание структуры проекта, установка зависимостей
2. **[Конфигурация](./02-configuration.md)** — настройка переменных окружения и конфигурационных файлов
3. **[Middleware](./03-middleware.md)** — настройка промежуточного ПО (безопасность, логирование, валидация)
4. **[Роутинг](./04-routing.md)** — создание маршрутов и обработчиков запросов
5. **[Работа с БД](./05-database.md)** — настройка Prisma и работа с базой данных
6. **[Запуск и деплой](./06-launch.md)** — запуск сервера и развертывание

## Быстрый старт

```bash
# 1. Установка зависимостей
npm install

# 2. Настройка .env файла
cp .env.example .env

# 3. Инициализация БД
npx prisma migrate dev --name init

# 4. Запуск сервера
npm run dev
```

## Структура проекта

```
shop_backend/
├── src/
│   ├── index.ts            # Точка входа: приложение, маршруты, запуск сервера
│   ├── config/
│   │   └── env.ts          # Валидация переменных окружения
│   ├── lib/
│   │   ├── logger.ts       # Настройка логирования
│   │   └── prisma.ts       # Подключение к БД
│   ├── middleware/         # Промежуточное ПО
│   ├── routes/             # Маршруты API
│   └── schemas/            # Схемы валидации (Zod)
├── prisma/
│   └── schema.prisma       # Схема базы данных
└── docs/                   # Документация
```

## Принципы

- **Минимум кода** — каждый пример содержит только необходимый код
- **Самостоятельность** — каждый раздел можно использовать независимо
- **Проверяемость** — все примеры проверены на работоспособность
- **Указание местоположения** — для каждого файла указан путь
